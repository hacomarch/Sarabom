# Sarabom 🏠

> **살아봄** - 실제 거주자와 연결되는 부동산 정보 공유 플랫폼

## 📌 프로젝트 소개

**Sarabom**은 부동산을 알아볼 때 가장 신뢰할 수 있는 정보원인 **실제 거주자**와 직접 소통할 수 있는 채팅 플랫폼입니다.

### 🎯 핵심 가치

부동산 앱이나 중개사를 통해서는 알 수 없는 **진짜 생생한 정보**를 얻을 수 있습니다:
- 🔊 실제 소음은 어떤가요?
- 🌞 채광은 충분한가요?
- 🚇 출퇴근 시간은 실제로 얼마나 걸리나요?
- 🏪 주변 생활 인프라는 편리한가요?
- 👥 이웃 분위기는 어떤가요?

### ✨ 주요 특징

- **직접 소통**: 관심 있는 집에 실제 살고 있는 사람에게 직접 질문
- **신뢰성**: 중개사나 리뷰가 아닌, 실거주자의 생생한 경험
- **실시간 채팅**: 궁금한 점을 즉시 물어보고 답변 받기
